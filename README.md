# RAG-Powered-Document-Chatbot
Built a RAG-based chatbot using Python, LangChain, and Streamlit to query PDFs, CSVs, DOCX, and TXT files with context-aware responses. Integrated GPT-4, Gemini, and Hugging Face models with configurable settings. Implemented full pipeline with ingestion, chunking, Chroma DB, retrieval, and memory.
